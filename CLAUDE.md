# manghe3

## gstack 配置

gstack 是一个快速无头浏览器，用于 QA 测试、站点试用和网页浏览。

### 规则

- 使用 gstack 的 `/browse` 技能进行所有网页浏览
- 切勿使用 `mcp__claude-in-chrome__*` 工具

### 可用技能

- `/browse` — 浏览网页、与元素交互、验证状态、截图
- `/qa` — QA 测试
- `/qa-only` — 仅运行 QA 测试
- `/connect-chrome` — 连接到 Chrome 浏览器
- `/setup-browser-cookies` — 配置浏览器 cookies
- `/review` — 审查代码
- `/ship` — 发布
- `/investigate` — 调查问题
